# **App Name**: Tailwind Canvas

## Core Features:

- Blank Canvas: Provide a blank Next.js and Tailwind CSS project setup for manual code input.
- Live Reloading: Enable live reloading during development to instantly view changes.
- Dependency Management: Automatically manage project dependencies with npm or yarn.

## Style Guidelines:

- Primary color: Deep indigo (#3F51B5) to reflect focus and development.
- Background color: Light gray (#F5F5F5) to provide a neutral canvas.
- Accent color: Teal (#009688) for interactive elements and highlights.
- Body and headline font: 'Inter' for a modern, clean, and readable look. 
- Minimal, functional icons used sparingly for key actions.
- Clean and simple layout to highlight user-generated content.
- Subtle animations for transitions and feedback to enhance the user experience.
